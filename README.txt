vivocyg 2.0

PURPOSE
Run cygwin for vivo machine

USAGE
Change c:\cygwin\vivocyg\command.txt to suit your purposes. Deleting it will default the command to:
start c:\\cygwin\\bin\\mintty -i c:\\cygwin\\vivocyg\\Cygwin-vivo.ico /bin/ssh mcarter@192.168.0.13


INSTALLATION

Accept the defaults - the installation directory is fixed as
c:\cygwin\vivocyg


Current executable at 18-Oct-2016 is at (Samsung):
 F:\windows-software\vivocyg-setup-2.0.exe
